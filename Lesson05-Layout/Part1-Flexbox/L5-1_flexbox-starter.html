👩‍💻 Code-Along #1: Horizontal Navigation

GOAL: Make these links display horizontally

  • Home
  • About
  • Services
  • Contact

📝 Instructions:

  1. Find the .demo-nav ul rule in your CSS
  2. Add display: flex;
  3. Add justify-content: center;
  4. Add gap: 30px;

👩‍💻 Code-Along #2: Perfect Centering

GOAL: Center this content both horizontally AND vertically

Welcome to Flexbox!

The modern way to build layouts

Third item in the Flexbox!

📝 Instructions:

  1. Find the .hero-box rule in your CSS
  2. Add display: flex;
  3. Add flex-direction: column;
  4. Add justify-content: center;
  5. Add align-items: center;

💡 TIP: flex-direction: column makes items stack vertically!

👩‍💻 Code-Along #3: Responsive Card Layout

GOAL: Make cards sit side-by-side and wrap on small screens

📦 Container

The parent that gets display: flex

🎯 Items

The children that become flexible

📱 Responsive

Adapts to different screen sizes

📝 Instructions - Part A (Container):

  1. Find the .cards-container rule
  2. Add display: flex;
  3. Add justify-content: center;
  4. Add flex-wrap: wrap; (makes it responsive!)
  5. Add gap: 20px;

💡 TIP: flex-wrap: wrap allows cards to drop to next line!

👩‍💻 Code-Along #4: Make Cards Equal Width

GOAL: Cards should grow to fill available space equally

⚡ Fast

Lightning quick performance

🔒 Secure

Bank-level encryption

💚 Easy

Simple to use interface

📝 Instructions - Part B (Items):

  1. Go back to the .card rule
  2. Add flex: 1; (makes all cards equal width!)
  3. Add min-width: 250px; (prevents cards from getting too narrow)

💡 TIP: flex: 1 makes items share space equally!

Then do the same for feature boxes:

  1. Find the .feature-row rule and add flexbox properties
  2. Find the .feature-box rule and add flex: 1;
